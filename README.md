## Overview
This lab demonstrates the implementation of modern CSS layout techniques using Flexbox and Grid, along with CSS transitions for interactive effects.

## Features Implemented

### Flexbox
- Horizontal navigation layout
- Equal-width column layout with reordering
- Centered card layout with tag styling

### CSS Grid
- Four-column responsive grid layout
- Overlapping grid elements using grid positioning
- Multi-row, multi-column structured layout using fractional units

### CSS Transition
- Smooth hover transition effects
- Different timing durations for hover-in and hover-out states
